# 项目结构
```
  assets  静态资源
  mock        测试数据
  api        请求接口
  components  共用组件
  router      路由
  store       vuex
  views      页面
  App.vue    入口
  man.js     入口
```
## 依赖
```shell
  npm i element-ui -S
  npm install axios
  npm install lodash -S
  npm i svg-sprite-loader -D
  npm install nprogress
```
